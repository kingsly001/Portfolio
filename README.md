# Kingsly Robbin | Personal Portfolio

A dynamic, responsive, and visually engaging personal portfolio website designed to showcase my skills, projects, and journey as a Full-Stack Developer. 

Built from scratch with a focus on clean UI/UX, thoughtful interactions, and modern web development practices.

## 🚀 Live Preview
*(https://kingslyrobbin.vercel.app)*

## ✨ Features

* **Dark & Light Mode:** Seamlessly toggle between visual themes using a custom UI switch.
* **Audio Integration:** Optional ambient background music with a custom play/pause toggle.
* **Custom Interactive Cursor:** A unique, stylized cursor that reacts and scales when hovering over clickable elements.
* **Eye-Tracking Avatar:** A fun, interactive character in the footer whose eyes follow your mouse movements across the screen.
* **Scroll Animations:** Smooth reveal animations triggered on scroll using the AOS (Animate On Scroll) library.
* **Fully Responsive Design:** A custom mobile-first layout featuring a hamburger menu and optimized viewing for all screen sizes.
* **Scrollspy Navigation:** Navigation links automatically highlight based on the section currently being viewed.
* **Preloader:** A clean, animated loading screen while the DOM and assets initialize.

## 🛠️ Built With

* **HTML5:** Semantic markup and structure.
* **CSS3:** Custom styling, CSS variables for theming, flexbox/grid layouts, and keyframe animations.
* **Vanilla JavaScript (ES6+):** DOM manipulation, event handling, math logic for the eye-tracking, and state management for settings.
* **AOS Library:** For scroll-triggered animations.
